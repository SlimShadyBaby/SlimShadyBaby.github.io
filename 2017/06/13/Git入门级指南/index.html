  • 拉取远端分支的代码

  • 把本地分支代码推送到远端分支上

TIPS：
1. git commit 和 git add可以合并为一个命令：
git commit -a -m ‘’ = git add . + git commit -m ‘’
2. git pull 可以理解为 git fetch + git merge:
git pull origin [分支名] = git pull is shorthand for git fetch followed by git merge FETCH_HEAD
3. 当推送的远端分支名和本地分支名一样的时候：
git push origin [分支名] = git push origin [本地分支]:[远端分支]
